IP Lookup API Documentation

Getting Started

  • Base URL: http://your-api-domain/api/ip
  • Authentication: None required
  • Rate Limiting: 50 requests per hour per IP

Endpoints

/api/ip/{ip}

Method: GET

Description: Retrieves information for the specified IP address.

Parameters:

  • Path Parameter:
    • ip (string, required): The IP address to query.
  • Query Parameters:
    • output (string, optional): Specifies the output format (json - default, xml, csv).
    • fields (string, optional): Comma-separated list of fields to include in the response.
    • callback (string, optional): Function name for JSONP callback.

Response Fields

Field Description
ipQueried IP address
successIndicates if the query was successful
messageResponse message (only present if success is false)
typeIPv4 or IPv6
continentContinent name
continent_codeContinent code
countryCountry name
country_codeCountry code
regionRegion name
region_codeRegion code
cityCity name
coordinatesLatitude and longitude, formatted as latitude, longitude
postalCodePostal code
callingCodeCountry calling code
timezoneIDTime zone identifier
ispInternet Service Provider name
orgOrganization name
asnAutonomous System Number and associated organization
mobileWhether the IP is associated with a mobile network
proxyWhether the IP is a known proxy
hostingWhether the IP is associated with a hosting provider
vpnWhether the IP is a known VPN
torWhether the IP is a known Tor exit node
utcCurrent UTC timestamp

Usage Examples

Basic Request

http://your-api-domain/api/ip/8.8.4.4

Filtered Fields

http://your-api-domain/api/ip/8.8.4.4?fields=ip,country,city

XML Output

http://your-api-domain/api/ip/8.8.4.4?output=xml

CSV Output

http://your-api-domain/api/ip/8.8.4.4?output=csv

JSONP Callback

http://your-api-domain/api/ip/8.8.4.4?callback=getIPinfo
